# @react-slices/components

React helper components for small functionalities and some CSS interactive modified HTML components

-   `CopyToClipboardIcon`

```
<CopyToClipboardIcon
   text={'text to copy onclick'}
   containerClass='abc'
   iconClass='cba'
   style={{color:'red'}}
   inline={true|| false}
/>
```

Example with children

```
<CopyToClipboardIcon
    text={'text to copy onclick'}
    containerClass='abc'
    iconClass='cba'
    >
    Copy Text or your icon
</CopyToClipboardIcon>
```
